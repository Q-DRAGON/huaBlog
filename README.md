# huaBlog
前端原生 JS 编写，原生 BOM 实现单页 SPA 应用, 后端用 node.js + express.
没有买服务器，所以没有预览，只有源码。
# 运行
1. 下载 node.js
http://nodejs.cn/download/
2. 安装 express:
` npm install express --save`
3.安装 body-parser:
`npm install body-parser --save`
4.安装 cookie-parser:
`npm install cookie-parser --save`
5.安装 multer:
`npm install multer --save`
6.命令行进入博客的目录, 启动:
`node appBlog.js`
